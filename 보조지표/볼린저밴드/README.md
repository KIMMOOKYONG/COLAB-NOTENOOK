# 볼린저 밴드
- 주가는 이동평균선을 기준으로 표준편차 범위 안에서 움직이다 평균으로 회귀한다고 가정
- 이동평균과 표준편차를 사용해서 주가의 높고 낮음을 상대적으로 정의
- 중심선: 20일 종가 단순 이동 평균선
- 상단선: 중심선 + 2 * (20일 종가 표준편차)
- 하단선: 중심선 – 2 * (20일 종가 표준편차)
